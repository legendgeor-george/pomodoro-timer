# Project: default

This is the workspace for the **default** project built by the Open Claw AI team.

## Rules
- All source code goes in src/
- All tests go in tests/
- All documentation goes in docs/
- Public/static assets go in public/
- Do NOT modify files outside this workspace directory
- Do NOT install global packages (use local installs: npm install, pip install)
- Use relative paths only
- Write .env.example with placeholder keys (NEVER real keys)
- Always add .env to .gitignore
