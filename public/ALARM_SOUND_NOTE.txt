ALARM SOUND FILE NEEDED
========================

To complete the Pomodoro Timer setup, please add an alarm sound file:

File name: alarm.mp3
Location: /public/alarm.mp3

Recommended sources for free alarm sounds:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/alarm/
- https://pixabay.com/sound-effects/search/alarm/

Requirements:
- Duration: 0.5-2 seconds
- Format: MP3
- Volume: Moderate (not too loud)
- Type: Simple beep or chime sound

Alternative: Generate using Web Audio API
You can generate a simple beep sound programmatically if preferred.

Note: The application will function without the alarm.mp3 file, but the audio
notification will not work (only browser notifications will be shown).
